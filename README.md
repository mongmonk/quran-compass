# Quran Compass

Quran Compass adalah aplikasi pembaca Al-Quran yang komprehensif dan mudah digunakan, dirancang untuk memberikan pengalaman membaca dan navigasi yang optimal bagi pengguna.

## Fitur Utama

- **Navigasi Surah yang Mudah**: Telusuri daftar surah dengan antarmuka yang bersih dan intuitif.
- **Tampilan Ayat yang Jelas**: Membaca ayat-ayat Al-Quran dengan font yang nyaman di mata.
- **Informasi Surah**: Dapatkan informasi detail mengenai setiap surah.
- **Mode Gelap/Terang**: Sesuaikan tampilan aplikasi sesuai dengan preferensi Anda.
- **PWA (Progressive Web App)**: Dapat diinstal di perangkat Anda untuk akses cepat.

## Teknologi yang Digunakan

- **React**: Library JavaScript untuk membangun antarmuka pengguna.
- **Vite**: Build tool yang sangat cepat untuk pengembangan web modern.
- **Tailwind CSS**: Framework CSS utility-first untuk desain yang responsif.
- **TypeScript**: Menambahkan tipe statis ke JavaScript untuk kode yang lebih aman.
- **Firebase**: Digunakan untuk hosting dan layanan backend lainnya.

## Cara Menjalankan Secara Lokal

**Prasyarat:** Node.js

1. **Clone Repository:**
   ```bash
   git clone https://github.com/mongmonk/quran-compass.git
   cd quran-compass
   ```

2. **Instal Dependensi:**
   ```bash
   npm install
   ```

3. **Jalankan Aplikasi:**
   ```bash
   npm run dev
   ```

4. **Build untuk Produksi:**
   ```bash
   npm run build
   ```

## Lisensi

Aplikasi ini dikembangkan untuk tujuan pendidikan dan spiritual. Silakan gunakan dan kembangkan lebih lanjut.
